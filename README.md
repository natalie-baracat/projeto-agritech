# PROJETO FINAL DO 2º SEMESTRE DE SENAI

O Estoque AgriTech (ou AgroTech) foi um projeto de encerramento de semestre desenvolvido pela segunda turma do SENAI (integrado ao SESI), em seu 2º semestre de aula no ano de 2024. Os alunos, sob a tutela dos professores [Douglas](https://github.com/douglas-camata) e [Ricardo](https://github.com/diasricardo), foram dividos em grupos; cada grupo possuia as mesmas orientações: utilizar o que foi estudado em sala para criar um **sistema de estoque** para uma empresa ficcional de insumos agrícolas.

O nosso sistema conta com uma landing page expositiva para qualquer um que entrar no site, login para administradores e responsáveis, CRUD de itens do estoque, tabela de movimentações (entrada e saída) de produtos, e controle de quantidade no estoque.

O projeto foi aprovado com nota máxima (100/100) após apresentações para professores, coordenadoras, pais, e demais equipes.
